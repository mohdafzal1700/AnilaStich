# Anila Stitching Center - React Vite

A modern, responsive website for Anila Stitching Center built with React, Vite, and Tailwind CSS.

## Features

- **React 18** - Latest React version with hooks
- **Vite** - Lightning-fast build tool and dev server
- **React Router** - Client-side routing
- **GSAP** - Smooth scroll animations
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **Responsive Design** - Mobile-first approach

## Project Structure

```
src/
├── App.jsx                 # Main app with routing
├── main.jsx               # Entry point
├── index.css              # Global styles with Tailwind
├── pages/
│   ├── Home.jsx          # Home page
│   ├── Services.jsx      # Services page
│   └── Contact.jsx       # Contact page
├── components/
│   ├── Navigation.jsx    # Navigation bar
│   └── Footer.jsx        # Footer component
```

## Getting Started

### Installation

```bash
npm install
# or
pnpm install
```

### Development

```bash
npm run dev
# or
pnpm dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
# or
pnpm build
```

### Preview

```bash
npm run preview
# or
pnpm preview
```

## Technologies Used

- **React 18.3.1** - UI library
- **React Router 6.21.0** - Routing
- **Vite 5.0.8** - Build tool
- **Tailwind CSS 4.0.0** - Styling
- **GSAP 3.12.2** - Animations
- **Lucide React 0.408.0** - Icons

## Animations

The site includes smooth GSAP animations for:
- Hero section fade-in effects
- Service cards stagger animations
- Scroll-triggered animations for benefits and info sections
- Contact page animations

## Styling

Uses Tailwind CSS with custom design tokens for:
- Color system (primary, secondary, accent, etc.)
- Responsive typography
- Spacing and layout utilities
- Dark mode support (included in CSS variables)

- Firefox (latest)
- Safari (latest)
